## Tasks

run a task:

```
npx hardhat --network <netowrk> <taskName> [taskArgs]
```

example:

```
npx hardhat --network bscTestnet OrangeAvatar:deploy
```

### OrangeAvatar

#### deploy

```
npx hardhat --network bscTestnet OrangeAvatar:deploy
```

#### mint

mint nft to a list of address

```
npx hardhat --network bscTestnet OrangeAvatar:mint <address1, addres2, addres3...>
```
